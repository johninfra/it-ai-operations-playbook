# Purpose

This document contains a collection of standardized AI prompts used to support IT Operations, Help Desk, Systems Administration, Cybersecurity, Documentation, and Technical Research activities.

The objective is to provide consistent, repeatable, and effective prompts that improve productivity and reduce time spent on routine tasks.

---

# AI Usage Guidelines

## Always Verify

AI generated content should always be reviewed before implementation.

Verify:

- Technical accuracy
- Security implications
- Organizational compliance
- Operational impact

---

## Never Blindly Implement

AI recommendations should never be deployed without human validation.

Examples:

- Firewall changes
- Active Directory modifications
- Group Policy updates
- Security controls
